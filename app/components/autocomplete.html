Autocomplete Directive

Bossy-autocomplete creates input element that matches an input string to strings in a provided dictionary.

Usage

<bossy-autocomplete config="myConfigObj"></bossy-autocomplete>

Options

  • config - Object that specifies autocomplete configuration.
  • config.dict - Array of strings to be recognized.
  • config.maxCorrections - Maximum number of edits to be applied when matching input. Defaults to 0.
  • config.maxSuggestions - Maximum number of suggestions offered for input. Defaults to 5.

Behavior

An input string will be matched with dictionary strings if:

  • A string in the dictionary starts with the input string.
  • The input string can be corrected to match a string in the dictionary within a specified number of edits.

Suggested matches are displayed below the input element and can be clicked to be applied to the input element.

Examples

  • 1. 50 States (no correction)


        myConfigObj = {
        dict: ['Alabama', 'Alaska', 'Wyoming']
    }

  • 2. 50 States (with correction)


        myConfigObj = {
        dict: ['Alabama', 'Alaska', 'Wyoming'],
        maxCorrections: 3,
        maxSuggestions: 8
    }
